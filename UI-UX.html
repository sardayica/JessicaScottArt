User Interface and Experience

ISTID Website Redesign(2024)

Made with Figma and Squarespace

While studying abroad, my program had groups work with the local ice cream company ISTID to redesign their website. Using Figma for the intial layout and navigation, and then remaking it in Squarespace. We focused on SEO performance, updating the overall look and media, having a Danish and English version of the site, and bettering our skills in prototyping and user testing.

Prototype for the Red Cross Parat Volunteer Site(2024)

Made with Microsoft Power Platform

While studying abroad, my program had groups work with the Red Cross to create an updated system for their Volunteers Events. We made the website have a face for volunteers, event organizers, and Red Cross administrators. Using the Microsoft Power Apps to create the front and back end of the website, we connected Power Automate to send out the approiate emails, as well as Power BI for data analysis.
